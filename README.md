# Requirements
1. Install [`nvm`](https://github.com/nvm-sh/nvm#installing-and-updating)
2. Install `node` with `nvm`
```
$ nvm install 18.17.1
$ nvm alias default 18.17.1
# or
$ nvm use
```
3. Run `cp .env.example .env` 

# Run
```
$ npm run test
```
Check in the [`package.json`](./package.json) for another commands.
