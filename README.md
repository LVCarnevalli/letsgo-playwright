# Requirements
1. Install `node` with [`nvm`](https://github.com/nvm-sh/nvm#installing-and-updating) or run `nvm use`
2. Run `cp .env.example .env` 

# Run
```
$ npm run test
```
Check in the [`package.json`](./package.json) for another commands.
